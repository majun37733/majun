# 多目标物品自动计数与位置标注系统

> 机器视觉与图像处理课程大作业

## 📌 项目简介

本项目实现了基于经典图像处理算法的**多目标物品自动计数与位置标注系统**。系统能够自动检测图像中的多个散落物品，统计数量，并通过矩形框标注每个物品的位置，同时输出各物品的中心坐标。

对真实拍摄的 14 个**药片**图像进行测试，系统成功检测并标注了全部目标，**准确率达到 100%**。

## 🧠 算法流程

```
原始图像 → 灰度化 → 高斯滤波降噪 → Otsu二值化 → 形态学处理 → 连通域分析 → 结果标注
```

### 核心算法说明

| 步骤 | 方法 | 说明 |
|------|------|------|
| 灰度化 | 加权平均法 | `Gray = 0.299×R + 0.587×G + 0.114×B` |
| 降噪 | 高斯滤波 | 15×15 核，σ=5.0 |
| 二值化 | Otsu 自适应阈值 | 自动计算最优分割阈值 |
| 形态学处理 | 开运算 + 闭运算 | 去噪 + 填充空洞 |
| 目标检测 | 连通域分析 | 按面积筛选有效目标 |

### 关键参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `gaussianKernelSize` | 15 | 高斯滤波核大小 |
| `gaussianSigma` | 5.0 | 高斯滤波标准差 |
| `minArea` | 3000 | 最小目标面积（像素） |
| `maxArea` | 200000 | 最大目标面积（像素） |
| `morphOpenIter` | 2 | 开运算迭代次数 |
| `morphCloseIter` | 3 | 闭运算迭代次数 |

## 📝 实验结论

1. **预处理是关键**：高斯滤波和形态学处理显著提升了检测精度
2. **Otsu 方法优势**：自适应阈值避免了手动调参，对光照变化有较好鲁棒性
3. **参数需调优**：不同场景下需调整滤波核大小、面积阈值等参数
4. **局限性**：对粘连目标和复杂背景的适应性有限，可结合深度学习方法改进
